<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository and install frontend dependencies</p>

```bash
npm install
```

<p>2. Run frontend</p>

```bash
npm run dev
```

<p>3. Install backend dependencies</p>

```bash
cd ayushayur-backend-master
npm install
```

<p>4. Run backend</p>

```bash
node server.js
```
